01 / Overview

Project Overview

LocaXion proposes a Hot Mill scrap tracking solution that consolidates location, weight, and grade into a single operational picture. Camera-based RTLS continuously tracks four forklifts and ties their movements to QR-coded scrap bins; three slitter-side operator stations integrate live AS400 production data and pre-empt scrap-grade mixing; two scale-side IoT modules publish weight in real time. All datapoints converge on an on-premise LocaXion server, which renders shift-floor dashboards for operators and KPI dashboards for executive review.

03 / Components

System Components

Four instrumented asset classes feeding a single on-premise consolidation layer. Each component carries its own role in the data flow; together, they close the loop between physical movement and operational visibility.

Forklifts — tracked & operator-aware

Camera-based RTLS for continuous, infrastructure-light location tracking, plus in-cab operator screens showing live scrap grade, bin ID, and weight.

4 units · RTLS · Operator UI · QR

Slitters — production-aware stations

Operator screen at each slitter, surfacing live production data direct from AS400 with alerts before scrap-grade mixing occurs.

3 stations · AS400 · Mix prevention

Weight scales — real-time telemetry

Two IoT modules instrument existing scales for continuous weight capture and per-bin history for operations and reconciliation.

2 scales · IoT · Weight telemetry

On-premise server — dashboards

Hosted within Arconic's network, consolidating scrap location, weight, and grade into shift-floor and executive KPI dashboards.

Server · Dashboards · KPIs
04 / Scope

Project Scope

The deployment is scoped to the Hot Mill, the asset population identified above, and the manufacturing-systems integrations needed to close the data loop.
